### Clone this project
```bash
git clone https://github.com/kkp785216/vegamovie.git
```

### Run this app
```bash
npm run start
# or
yarn start
```

Open [http://localhost:3000/vegamovie](http://localhost:3000/vegamovie) with your browser to see the result.
